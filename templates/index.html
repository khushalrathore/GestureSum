Getting Started

  1. 1. Go to "Download ASLI" button (the top right corner).
  2. 2. Locate the downloaded .zip file and extract it.
  3. 3. Open the extracted folder in VSCode or any IDE.
  4. 4. Run the following command in your IDE Terminal (within the root folder) :
  5. > pip install -r requirements.txt
  6. Or
  7. > pip3 install -r requirements.txt
  8. based on your pip configuration.
  9. 5. Once all the required packages are downloaded run this app using :
  10. > python index.py
  11. Or
  12. > python3 index.py
  13. based on your python configuration.
  14. 6. Voila! It's running on localhost.
